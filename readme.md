# textbox

PHPで実装された、プライベート環境向けQiita系サイトです

## バージョン

0.1.0

## 眺める

http://textbox-demo.nijibox.net/ にあります

* User: user1@example.com
* Pass: password1

## 動かす

### 必要環境

* Laravel 5.2系が動くPHP
* RDBサービス(MySQL, SQLite)

### 動かし方

※ローカル環境向け

```bash
$ composer install
$ php artisan migrate
$ php artisan serve
```
